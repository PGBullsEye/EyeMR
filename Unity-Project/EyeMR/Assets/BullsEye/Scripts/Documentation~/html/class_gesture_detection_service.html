Detailed Description

This service, if once attached to a GameObject, communicates with the FixationDetectionService. InteractionTechniques can subscribe to the service. A script can do this by implementing the IGestureInteractionTechnique interface. When subscribing a script must deliver a gesture to define which gesture must occur to trigger the script. From now on the service will check every time it gets notified about a new fixation if a gesture of any subscribed script has occurred. If that is the case the service will trigger the IGestureInteractionTechnique.OnGesture method of the corresponding object.

Parameters

errorAngleThe maximal error between angles defined in the gesture and occurring angles. Percentage of 360 degrees (e.g. 0.1 corresponds to degrees). Default value is 0.05.
